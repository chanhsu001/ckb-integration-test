# ckb-bench

In general, ckb-bench is a transaction generator. It can be used to benchmark transactions on CKB nodes.

We measure CKB network's transaction throughput with ckb-bench. Refer to [this document](./docs/usage.md).

CKB-bench can be used for generating many transactions onto a CKB network, such as testnet. Refer to [bench-on-testnet](./docs/bench-on-testnet.md).
